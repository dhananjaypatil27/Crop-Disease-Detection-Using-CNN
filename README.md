
# Crop Disease Detection Using CNN

## Problem Statement

Crop diseases can reduce agricultural productivity and cause major losses to farmers. Early detection of plant diseases helps in taking timely action and improving crop yield. This project uses **Convolutional Neural Networks (CNN)** to automatically detect crop diseases from leaf images.

## Objective

The main objective of this project is to build a **deep learning model** that can identify whether a crop leaf is healthy or diseased using image classification techniques.

## Project Summary

* The project uses **image data of crop leaves** for disease detection.
* Images are preprocessed and resized before training the model.
* A **CNN model** is built to classify the images into healthy or diseased categories.
* The model learns patterns and features from leaf images to detect diseases accurately.
* The model is trained and evaluated using deep learning techniques.

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* OpenCV

## Model Development

* Image preprocessing and normalization
* CNN model architecture creation
* Model training and validation
* Performance evaluation using accuracy and loss metrics

## Outcome

The CNN model can automatically detect crop diseases from leaf images, which can help farmers identify plant diseases early and take preventive measures to protect crops and improve agricultural productivity.
